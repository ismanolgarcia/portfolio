# Portfolio Website

This is my personal portfolio website built using HTML, CSS, JavaScript, and Sass. I have also used Font Awesome for icons and AOS for scroll animations.

## Installation

1. To run this project locally, follow these steps:

   `https://github.com/ismanolgarcia/portfolio.git `

2. Clone the repository to your local machine.

3. Open the project directory in your code editor.

4. Install the necessary dependencies using npm install.
   `npm install `

5. Script to run sass.
   `npm run sass`

Usage
This portfolio website showcases my work, skills, and experiences as a web developer. It is designed to be user-friendly and easy to navigate. The website is divided into several sections including Home, About, Skills, Projects, etc...

###### Libraries you use

- [Font Awesome](https://fontawesome.com/)
